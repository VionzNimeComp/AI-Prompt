## 🎀 Tentang Project

> Website ini menggunakan teknologi AI untuk mengubah gambar menjadi prompt yang dapat digunakan untuk berbagai keperluan kreatif.

**AI Prompt • ShiroNekoAI Art** adalah platform berbasis web yang memungkinkan Anda mengunggah gambar dan mendapatkan prompt AI yang mendetail dan akurat. Dibangun dengan ❤️ oleh **ShiroNeko** untuk membantu kreativitas Anda.

---

## ✨ Fitur Utama

| Fitur | Keterangan |
|-------|------------|
| 🖼️ **Upload Gambar** | Drag & drop atau klik untuk upload |
| 🤖 **Generate Prompt** | AI otomatis menghasilkan prompt dari gambar |
| 📋 **Copy Prompt** | Salin prompt dengan satu klik |
| 🎨 **Responsive Design** | Tampilan optimal di semua perangkat |
| 🌙 **Dark Mode** | Tampilan gelap yang elegan |
| 📱 **Mobile Friendly** | Dapat diakses dari HP |
| ⚡ **Cepat & Ringan** | Performa optimal |

---

## 🚀 Cara Menggunakan

### 1️⃣ Upload Gambar
Klik area upload atau drag & drop gambar yang ingin diubah menjadi prompt.

### 2️⃣ Generate Prompt
Tekan tombol **"⚡ Generate Prompt"** dan tunggu beberapa detik.

### 3️⃣ Copy & Gunakan
Prompt akan muncul di bawah, klik **"📋 Copy"** untuk menyalinnya.
